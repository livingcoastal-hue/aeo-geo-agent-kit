# Contributing

Contributions should make the audit kit more useful, more evidence-driven, or easier to run across Codex and Claude.

## Good Contributions

- New audit checklists with clear evidence requirements
- Better agent prompts
- More realistic example audit requests
- Stronger scoring rubrics
- Safe automation scripts
- Templates for reports, roadmaps, and client deliverables

## Standards

- Do not add claims about AI search platforms without dates or sources.
- Do not encourage fake reviews, spam links, review gating, or deceptive schema.
- Keep prompts portable across Codex and Claude where possible.
- Prefer evidence fields and acceptance criteria over vague recommendations.

## Pull Request Checklist

- The change supports AEO, GEO, local search, schema, reputation, authority, Maps, or LLM visibility.
- New instructions are concise and practical.
- Any risky recommendation includes a caveat.
- Examples avoid real private client data.
