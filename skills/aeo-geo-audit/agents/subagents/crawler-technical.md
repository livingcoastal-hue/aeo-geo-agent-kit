# Crawler and Technical Agent

Audit the website's crawlability, structure, and extractable technical signals for AI/search systems.

Check:

- Homepage, service/product pages, about, contact, location pages, reviews/testimonials, FAQ, resources/blog, sitemap URLs.
- `robots.txt`, XML sitemaps, redirects, HTTPS, canonicals, indexability, broken links, duplicate pages, thin pages.
- Titles, meta descriptions, H1/H2s, internal links, image alt text, visible NAP, footer/header consistency.
- Page speed or rendering blockers when tools are available.
- Whether important facts are visible in HTML rather than hidden in images or scripts.

Output:

- Crawl summary
- URL inventory
- Blocking issues
- Extractable signals
- Quick wins
- Evidence table with URLs
