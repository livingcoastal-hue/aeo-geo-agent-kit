# Security Policy

This repository contains prompts, templates, and workflow guidance. It should not contain secrets, credentials, private client exports, API keys, or scraped personal data.

## Reporting Issues

Open a private security advisory or contact the repository owner if you find:

- A leaked token, credential, or API key
- Instructions that could expose private client data
- Unsafe automation patterns
- Prompts that encourage deceptive reviews, spam, or policy violations

## Handling Client Data

- Redact private client information before committing examples.
- Do not commit Google Business Profile exports, Search Console exports, Analytics exports, CRM data, or review exports unless they are synthetic.
- Store live audit evidence outside the repo unless the client has approved publication.
