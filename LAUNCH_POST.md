# Launch Post

AI search is changing how customers discover local businesses.

I built `aeo-geo-agent-kit`: a Claude- and Codex-compatible skill and agent bundle for auditing whether a company is ready to be found, trusted, recommended, and cited across AI answers, Google Maps, local search, schema/entity systems, reviews, links, and LLM search experiences.

The kit includes:

- A Codex skill
- A Claude bridge skill
- A main coordinator agent
- Sub-agents for crawl, schema, content, Maps/NAP, reviews, links, and LLM visibility
- Scoring rubric
- Audit report template
- Remediation roadmap template
- Prompt library for AI visibility testing

The goal is simple: move beyond old keyword-only SEO audits and build a practical readiness system for AI-first discovery.
